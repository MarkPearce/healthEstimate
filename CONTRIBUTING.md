# Development

No build step. Symlink `src/` into your Foundry data folder:

```
ln -s "$(pwd)/src" "/path/to/FoundryVTT/Data/modules/healthEstimate"
```

Edits to JS, CSS, templates, or lang files are then live in Foundry — just refresh.
